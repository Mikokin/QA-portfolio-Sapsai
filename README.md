# QA Portfolio — Сапсай Олег

Добро пожаловать в моё портфолио тестировщика

Здесь представлены мои проекты, демонстрирующие практические навыки ручного и API-тестирования полученные в ходе работы, а также создании собственных pet-проектов.

Я развиваюсь в сфере информационных технологий и применяю полученные знания на практике. Обладаю опытом работы с инструментами JIRA, Azure DevOps, YouTrack, DevTools, Postman, Swagger, Android Studio, Git, а также владею базовыми навыками SQL, HTML, CSS, C#, JavaScript и Python.

Понимаю принципы клиент-серверного взаимодействия, архитектуру веб-приложений и особенности работы с REST и SOAP API.

Интересуюсь процессами разработки и тестирования программного обеспечения, активно изучаю подходы Agile, Scrum, тест-дизайн, а также расширяю свои компетенции в области автоматизации тестирования.
